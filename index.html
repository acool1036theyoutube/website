What is Scratch Addons?

Scratch Addons is an browser extension that provides new features and themes for the Scratch website and project editor. Scratch Addons has a mission mission to collect, archive, and put into use userstyles and userscripts, developed by several members of the Scratch community.

Features

Various features to empower your Scratch website and editor experience

Easy-to-use interface with various controls for each feature and theme

Free, open source, and maintained by many users from the Scratch community

Install

Chrome Firefox

Or load it locally.
